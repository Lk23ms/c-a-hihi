## Cách dùng repo này
- Đọc lý thuyết → chạy thử ví dụ code → tự làm bài tập gợi ý trước khi xem đáp án (nếu có).
- Mỗi bài đều có phần "Bài tập gợi ý" ở cuối để luyện tập.
- Trình biên dịch khuyến nghị: g++ (C++17 trở lên). Compile ví dụ:
  ```bash
  g++ -std=c++17 -O2 -o a.out file.cpp
  ./a.out
  ```
